憂鬱症的自我檢測

台灣人憂鬱症量表
病人健康問卷(PHQ-9)

請你根據最近一週以來身體與情緒真正的感覺勾選最符合的一項。


1.我覺得想哭

2.我覺得心情不好

3.我覺得比以前容易發脾氣

4.我睡不好

5.我覺得不想吃東西

6.我覺得胸口悶悶的 ( 心肝頭或胸坎綁綁 )

7.我覺得不輕鬆、不舒服 ( 不爽快 )

8.我覺得身體疲勞虛弱、無力( 身體很虛、沒力氣、元氣及體力 )

9.我覺得很煩

10.我覺得記憶力不好

11.我覺得做事時無法專心

12.我覺得想事情或做事時，比平常要緩慢

13.我覺得比以前較沒信心

14.我覺得比較會往壞處想

15.我覺得想不開、甚至想死

16.我覺得對什麼事都失去興趣

17.我覺得身體不舒服( 如頭痛、頭暈、心悸或肚子不舒服…等 )

18.我覺得自己很沒用



在過去兩個星期，有多少時候您受到以下任何問題所困擾？


1.我做事時提不起勁或沒有樂趣

2.我感到心情低落、沮喪或絕望

3.我入睡困難、睡不安穩或睡眠過多

4.我感覺疲倦或沒有活力

5.我食慾不振或吃太多

6.我覺得自己很糟、失敗，或讓自己或家人失望

7.我對事物專注有困難， 例如閱讀報紙或看電視

8. 我動作或說話速度緩慢，或煩躁或坐立不安

9. 我不如死掉或用某種方式傷害自己的念頭
